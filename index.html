CSS Layout Reference

Layout Rules

Rule Description
position property allows you to specify the position of an element.
relative position is relative to its default position on the page.
absolute element’s position is relative to its closest positioned parent element
fixed element’s position can be pinned to any part of the web page.
sticky element can stick to a defined offset position when the user scrolls its parent container.
z-index specifies how far back or how far forward an element appears on the page.
display allows you to control how an element flows vertically and horizontally in a document.
inline elements take up as little space as possible
block elements take up the width of their container
inline-block can have set (width) and (height), but they can also appear next to each other

Changing Box Model

Rule Discription/Values
box-sizing property controls the box model used by the browser.
Value 1 content-box
Value 2 border-box
